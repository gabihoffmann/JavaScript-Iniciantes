Manipulação de dados

Para o JS os dados armazedos via prompt são retornados como string, então é preciso converter para o tipo de dado desejado.
Para conversão de String para number podemos usar as funções
Number.parseInt(n) para inteiro,
Number.parseFloar(n) para ponto flutuante
Number(n) onde cria-se um objeto do tipo number, se o argumento não poder ser convertido retorna um NaN.
Para a conversão para dado do tipo String usa o String(s).

Formatação de strings e números

Com o ECMAScript 6 é possivel utilizar o Template string que são strings multilinha e interpolação de string. Os Template string são envolvidos por acentos grave `` e podem possuir placeholder ${}. Que apresenta um código mais limpo em comparação com a concatenação.
Exemplo:
console.log('Quinze é ' + (a + b) + ' e\nnão ' + (2 * a + b) + '.'); // concatenação
console.log(`Quinze é ${a + b} e não ${2 * a + b}.`);//Template string
s.length // comprimento da string
s.toUpperCase(); //Todas os caracteres MAIUSCULO
Com JS é possivel formatar números tambem, por exemplo para a quantidade de casas decimais apos a virgula utiliza number.toFixed(n), onde number é o valor formatado e n número de casas. Outro exemplo é possibilidade de formatar de acordo com a localização geografica da informação n.toLocaleString('pt-BR', {style: 'currency', currency:'BRL'})
