Operadores Aritméticos

Os operadores aritmericos no JS +(soma), -(subtração), *(multiplicação), /(divisão), %(resto divisão) e ** (potencia) são utilizados para realizar calculos. Todavia deve-se lembrar da ordem de precedência dos operadores:
() : expressoões entre parenteses
** : potenciação
* / % : mulltiplicação, divisão e modo
+ - : soma e subtração

Operadores de atribuição

A atribuição em JS é dada pelo simbolo =, onde um valor (no lado direito) é atribuido a um operando/variavel que está a esquerda. Dessa forma x = y, significa que está ocorrendo a atribuição do valor de y para x.
A auto atribuição é quando o valor recebe ele mesmo com alguma operação, exemplo atribuição com adição: x = x + y pode ser simplificado para x += y.
O incremento adiciona um ao operando ou seja z = z + 1 pode ser simplificado para z++ ou z--

Operadores Relacionais e Comparação

São os operadores utilizados para comparar valores e seu resultado é um valor booleano, os comparadores são maior >, menor <, maior igual >= e menos igual <=
No JavaScript temos dois tipos de comparação, a de igualdade e identidade, onde a comparação == e != compara o valor, na comparação estrita/identidade === e !== compara o tipo e valor
typeof 5 //number
typeof '5' //string
5 == 5 // true
5 == '5' // true
5 === 5 //true
5 === '5' //false

Operadores Lógicos

Os operadores lógicos do JS são ! negação, && conjunção (END), || disjunção (OR)

Operador Ternário

O operador ternário é uma expressão composta de três partes, onde o primeiro é o teste, o segundo é ação para quando o teste é verdadeiro e o terceiro para quando o teste é falso.
condição ? ifTrue : ifFalse
