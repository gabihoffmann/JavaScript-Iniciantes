Comentário

Um comentário é uma parte de código que não é interpretada pelo navegador. Pode ser utilizada, também, para colocar textos que ajudam a identificar o que está sendo feito naquele trecho de código. Esta é uma excelente prática. No JS comentários são definidos por barra duplas para linha de código e conjunto /* */ para um bloco de código.

exemplo de comentário : // window.alert("Este alerta não será mostrado");

exemplo de comentário :
/*
Exemplo de comentário para várias linhas.
Basta escolher o início e fim.
*/

Variáveis

São espaços delimitados de memória de computador para realizar o armazenamento de dados. No JavaScript existem tem três tipos de declaração de variaveis, vamos iniciar com a declaração do tipo var (variable statement) declara uma variável, opcionalmente é possível atribuir à ela um valor em sua inicialização. Sua sintaxe é dada por var identificador = valor; onde identificador é o que identifica uma variável e segue as seguintes regras:

  • pode começar com letras, $(cifrão) ou _(underline)
  • não pode começar com número
  • pode conter letras e números
  • pode conter acento e simbolo
  • não pode conter espaços
  • não pode ser palavra reservada: var, function, ...
  • maiúsculas e minúsculas fazem diferença (case-sensitive)

Tenha em mente que ao declarar uma variavel utilize nomes coerentes para as variaveis não seja um "programador contador que só declara n1,n2,n3".

Tipos de dados

O valor é o dado que será armazenado na variavel e esse dado possuem um tipo (data types), para saber o tipo de variavel utiliza o typeof variable, no JS temos variaveis dos tipos:

  • number
  • string
  • boolean
  • null
  • undefined
  • object
  • function
